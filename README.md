### AI & Machine Learning Projects

- **[Seismic-Waves Classification](https://github.com/alesanchezb/ondas_sismicas_delfin)** - Seismic wave analysis project applying AI techniques to study and predict dolphin behavior patterns.

- **[Classify the world](https://github.com/alesanchezb/clasifica-el-mundo)** - Image classification project using machine learning techniques to categorize different types of objects and scenes.

- **[Cleaning Data with CNN](https://github.com/alesanchezb/ai-cup-chatbot)** - Neural Network developed for the AI Cup competition to validate and ensure quality control of labeled images through a chat interface, preparing high-quality training data for future models.
  
- **[CSIRO - Image2Biomass Prediction and Predict the Introverts from the Extroverts](https://github.com/alesanchezb/biomass-and-introverts)** - Data science competition project analyzing biomass data and behavioral patterns.

- **[Ganadería en Sonora - Data Analysis](https://github.com/alesanchezb/ganaderia-sonora-dataton)** - Interactive Power BI dashboard developed for the 1st "Sonora en Datos" Datatón. Analyzes livestock production and economic gains across Sonora's municipalities using open government data, featuring interactive visualizations and key performance indicators.
